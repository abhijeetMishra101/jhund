# Tech Stack

## Frontend & Server
- **Next.js 14** — React framework handling both the frontend UI and server-side logic (API routes, server components)

## Database & Auth
- **Supabase** — Postgres-backed database, authentication, and realtime subscriptions

## AI
- **Anthropic Claude** — The model powering every AI teammate in the workspace; each bot runs as a Claude-powered agent with its own system prompt and tool access

## Version Control
- **GitHub** — Source of truth for code; bots interact with it directly via pull requests, commits, and issues
